スラッシュコマンド

スラッシュコマンド ドキュメントバージョン 2.2.2


前置詞と引数

スラッシュコマンドの接頭辞： /
必須の引数： <>
オプションの引数： [<>]

必須コマンド

すべてのカテゴリのリスト： /help
カテゴリー内のヘルプ： /help <カテゴリ名>

グラフ

カテゴリー 説明 コマンド
ファン ユニークで楽しいコマンドを持つカテゴリー 8ball, advice, booba, compliment, fact, insult, rickroll, roll, rps, say
インタラクション やりたいことを表現する bite, cuddle, highfive, hug, kiss, lick, pat, poke, punch, squish, yeet
アクション 感情を表現する angry, awkward, blush, cry, dance, eat, smile
ユーティリティ 真剣さだ！ avatar, discord, ping, poll
スプラトゥーン 2 & 3 スプラトゥーン2と3のコマンド s2, s3
ムラムラす アダルトコンテンツに関する警告 18歳以上 18歳以上入場可

ファン

/8ball <問題> エイトボールに聞いてみよう

/advice アドバイスをもらう

/booba ...

/compliment 褒めてもらう

/fact <カテゴリー> 特定のカテゴリーからファクトを取得する

/insult 侮辱される

/rickroll [<@user>] 誰かをRickrollする

/roll <番号> 特定の辺の数から数字を転がす

/rps <石/紙/はさみ> メイドとジャンケンをする

/say <テキスト> 荒らしに最適なボットとしての発言

インタラクション

/bite [<@user>] 誰かを噛む

/cuddle [<@user>] 誰かを抱きしめる

/highfive [<@user>] 誰かとハイタッチ

/hug [<@user>] 誰かを抱きしめる

/kiss [<@user>] キスをする

/lick [<@user>] 誰かをなめる

/bite [<@user>] 誰かを噛む

/pat [<@user>] 誰かパット

/poke [<@user>] 誰かを突く

/punch [<@user>] 誰かを殴る

/squish [<@user>] 誰かを潰す

/yeet [<@user>] 誰かを投げ飛ばす

アクション

/angry 怒る >:(

/awkward 気まずくなる >-<

/blush シャイ.. OvO

/cry 泣け！わあああ... D:

/dance ダンス ララララ！

/eat 食べること! ノムノム

/smile :)

ユーティリティ

/avatar user <@user> 誰かのアバターをゲット

/avatar server サーバーアイコンの取得

/discord ボットを招待するか、サポートサーバーに参加してください！

/ping Pong!

/poll <投票名> <一> <二> [<三>] [<四>] [<五>] 投票を作成する

スプラトゥーン 2

/s2 random <選択> 多くの選択肢からランダムに取得する

/s2 list <選択> すべてのオプションのリストを取得する

/s2 schedule <ゲームモード> ゲームモデモードのスケジュールを取得する

[サーモンランゲームモードオプションは一時的に無効化されました。］

スプラトゥーン 3

/s3 random <選択> 多くの選択肢からランダムに取得する

/s3 list <選択> すべてのオプションのリストを取得する

/s3 schedule <ゲームモード> ゲームモデモードのスケジュールを取得する

/s3 map <マップ名> <ゲームモード> <イメージタイプ> 選択肢から画像を取得する
